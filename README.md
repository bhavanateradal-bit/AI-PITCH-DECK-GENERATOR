AI Pitch Deck Generator

Project Description

The AI Pitch Deck Generator is a web application that helps startups, entrepreneurs, and students create professional pitch decks quickly using AI. It provides an easy-to-use interface for generating well-structured presentation content.

Features

- AI-powered pitch deck generation
- User Registration and Login
- Secure Authentication
- Profile Management
- Responsive and User-Friendly Interface
- Fast and Efficient Content Generation

Technologies Used

- HTML
- CSS
- JavaScript
- Bootstrap
- python
- Django Framework
- generative AI/Open AI
- MySQL
- SQLite(optional)

How to Run the Project

1. Download or clone this repository.
2. Open the project folder in your code editor.
3. Open the terminal in the project directory.
4. Run the required command to start the local development server.
5. Copy the server URL displayed in the terminal (for example, "http://localhost:8000").
6. Paste the URL into your web browser.
7. The application will open and be ready to use.
